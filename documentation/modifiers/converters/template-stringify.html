TemplateStringify Collection Converter

Converts the Declarations property of the specified PreModifierArgs into an instance of PostModifierArgs where the Emitted property contains the stringified HTML template. 

	abstract class TemplateStringify : SuperScript.Modifiers.Converters.CollectionConverter
	{
		abstract string CollateTemplates(IEnumerable<DeclarationBase> declarations);
		bool PreCompile { get; set; }
		string ScriptPath { get; set; }
		
		// Inherited members
		EmitMode EmitMode { get; set; }
		override PostModifierArgs Process(PreModifierArgs args)
	}

The reason for this being an abstract class is that the implementation of CollateTemplates will differ depending upon the client-side templating library.

Name Type Description
CollateTemplates IEnumerable<DeclarationBase>

Enumerates each TemplateDeclaration and converts the declarations into a string.

If PreCompile is true then pre-compilation should occur in this method.

Although the parameter accepts a generic collection of DeclarationBase implementations, only implementations which are of type TemplateDeclaration will be considered.

PreCompile bool

Gets or sets whether the templates should be pre-compiled before being sent to the client.

Optional. Default value is false.

ScriptPath string

Gets or sets the path for the script file.

Inherited members
Process PostModifierArgs

Converts the Declarations property of the specified PreModifierArgs into an instance of PostModifierArgs where the Emitted property contains the stringified HTML templates.

EmitMode EmitMode

Determines the mode in which this implementation emits contents.

Optional. Default value: EmitMode.Always

This implementation of CollectionConverter simply enumerates each instance of TemplateDeclaration in the PreModifierArgs.Declarations property and calls DeclarationBase.ToString().

The result of this conversion process is then passed to the Emitted property of a PostModifierArgs object.
