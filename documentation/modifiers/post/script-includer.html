ScriptIncluder Post-conversion Modifier

Optionally includes the client-side (typically JavaScript) HTML template framework script. 

	class ScriptIncluder : SuperScript.Modifiers.Post.CollectionPostModifier
	{
		override PostModifierArgs Process(PreModifierArgs args)
		string ScriptPath { get; set; }
	}
Name Type Description
Process IHtmlString

Prepends the contents of the referenced script file to the Emitted property of the PostModifierArgs object.

ScriptPath string

Gets or sets the path to the client-side (typically JavaScript) HTML template framework script to be packaged with the HTML templates.

Optional. Default value: false.

Like all implementations of CollectionPostModifier, ScriptIncluder exposes a public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of either

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.

The result of the Process method is an instance of PostModifierArgs where the Emitted property contains the output of the minification process.

Important: It should be remembered that multiple implementations of CollectionPostModifier may be configured for an IEmitter.
