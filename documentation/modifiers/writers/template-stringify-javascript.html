TemplateStringifyJavaScript HTML Writer

Writes the formatted output of the Emitted into an HTML tag.

This implementation of HtmlWriter is very similar to SuperScript.JavaScript.Modifiers.Writers.ScriptTagWriter. However, this class has the HTML tag hard-coded. 

	class TemplateStringifyHtml : SuperScript.Modifiers.Writers.HtmlWriter
	{
		EmitMode EmitMode { get; set; }
		override IHtmlString Process(PostModifierArgs args)
	}
Name Type Description
Process IHtmlString

Executes this implementation of HtmlWriter upon the specified argument object.

Inherited members
EmitMode EmitMode

Determines the mode in which this implementation emits contents.

Optional. Default value: EmitMode.Always

Like all implementations of HtmlWriter, TemplateStringifyHtml exposes a single public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of either

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.
