What Does It Do?

This library is an add-on for SuperScript. It is strongly recommended to read the documentation for the core project before proceeding with this library. If you haven't yet read the core documentation then it is recommended that you start by reading about the core concepts.

SuperScript.Templates adds functionality for declaring HTML template-specific entities in server-side .NET code.

While the SuperScript library contains the core functionality for making and emitting declarations, SuperScript.JavaScript contains JavaScript template-specific implementations of those core classes. Broadly speaking, SuperScript has three main aspects: declarations, which are instances of DeclarationBase, and emitters, which consist of a combination of modifiers.

SuperScript.Templates therefore consists of an implementation of DeclarationBase and modifiers which are specific to creating and consuming HTML templates.

Example

This library contains an implementation of DeclarationBase and several implemantations of ModifierBase. It should be remembered that multiple modifiers may be configured into a single IEmitter.
